
#  Projeto YOLO - Detecção de Vacas 

##  Descrição
Este projeto utiliza o modelo YOLOv8 para detecção de objetos (vacas) em imagens e vídeos.

##  Tecnologias utilizadas
- Python
- YOLOv8 (Ultralytics)

## Estrutura
dataset/
├── images/
├── labels/

runs/
├── detect/

train.py  
data.yaml  
best.pt  

## Resultados
O modelo foi treinado com mais de 20 imagens rotuladas.
O arquivo `best.pt` contém o modelo final.
